# k8s

## This repository hosts k8s .yaml files to configure clusters

### basic_config.yaml manifest 
This manifest configures a basic Deployment

### dashboard-ingress.yaml
This manifest configures a basic Ingress

### low-resource-range.yaml
This manifest set the limits for CPU and RAM usage

### low-usage-limit.yaml
This manifesst configures a Deployment with hardware limits

### myapp-secret-tls.yaml
This manifest configures a Secret

### nginx-deployment.yaml
This manifest configures a nginx Deployment and Pod

### tls-example-ingress.yaml
This manifest configures Ingress with TLS
